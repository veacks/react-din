# Product Summary

`@open-din/react` helps React developers build and inspect declarative Web Audio graphs through a reusable library and the DIN Editor.

## Priority Features

1. Public React audio components for graph composition.
2. Sequencing, transport, analysis, sources, and effects for musical workflows.
3. DIN Editor nodes that mirror the audio graph concepts and generate code.
4. Contributor governance that keeps documentation, test coverage, and node/component registration aligned.
